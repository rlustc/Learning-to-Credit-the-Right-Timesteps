# OTCA Homepage

This directory contains a static GitHub Pages homepage for **Learning to Credit the Right Steps: Objective-aware Process Optimization for Visual Generation**.

To publish it from an OTCA GitHub repository:

1. Push this `docs/` directory to the default branch.
2. Open `Settings` -> `Pages`.
3. Set `Source` to `Deploy from a branch`.
4. Select the default branch and `/docs`.
5. Save and wait for deployment.
